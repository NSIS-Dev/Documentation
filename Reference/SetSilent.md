# SetSilent

---

Sets the installer to silent mode or normal mode. See [`SilentInstall`][1] for more information about silent installations. Can only be used in [`.onInit`][2].

## Parameters:

	silent | normal

## History:

Added in NSIS v2.0 Beta 4
