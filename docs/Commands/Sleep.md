# Sleep

Pauses execution in the installer for `sleeptime_in_ms` milliseconds. `sleeptime_in_ms` can be a variable, e.g. "$0" or a number, i.e. "666".

## Parameters

    sleeptime_in_ms

## Example

    DetailPrint "sleeping..."
    Sleep 3000
    DetailPrint "back to work"

## History

Added in NSIS v1.1a
