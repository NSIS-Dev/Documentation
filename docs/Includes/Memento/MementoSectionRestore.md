# ${MementoSectionRestore}

Add a call to `${MementoSectionRestore}` to [`.onInit`][1] to restore the state of all sections from the registry.

## Syntax

    ${MementoSectionRestore}

## Example

    Function .onInit
        ${MementoSectionRestore}
    FunctionEnd
