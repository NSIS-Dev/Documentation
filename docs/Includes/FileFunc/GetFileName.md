# ${GetFileName}

Get last part from directory path.

## Syntax

    ${GetFileName} "[PathString]" $var

## Example

    Section
        ${GetFileName} "C:\Program Files\Winamp\uninstwa.exe" $R0
        ; $R0="uninstwa.exe"
    SectionEnd
