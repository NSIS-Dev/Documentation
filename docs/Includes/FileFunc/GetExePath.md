# ${GetExePath}

Get installer pathname ([`$EXEDIR`][1] with valid case for Windows 98/Me).

## Syntax

    ${GetExePath} $var

## Example

    Section
        ${GetExePath} $R0
        ; $R0="C:\ftp"
    SectionEnd
